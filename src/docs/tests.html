Code and Demos

How to highlight code and show previews or import iframes.

The 'lang' value can be 'html' 'js' 'scss' 'js'.

Inline

Between <pre></pre> and <code></code> there must be no space or newline.


<pre><code class="lang">
<!-- code here -->
</code></pre>

Demos

Below all the examples of demos, including how to show previews and import iframes.

Default markup for the demos, the attribute [data-lang="lang"] is for the code highlight.
<div class="demo"> <div class="demo-item"> <div class="demo-source" data-lang="html"> <!-- code here --> </div> </div> </div>
You can add a description by adding a .demo-text div with a descriptive text.
<div class="demo"> <div class="demo-item"> <div class="demo-text">Description</div> <div class="demo-source" data-lang="html"> <!-- code here --> </div> </div> </div>
Use the class .demo-preview to show the preview.
<div class="demo"> <div class="demo-item demo-preview"> <div class="demo-source" data-lang="html"> <!-- code here --> </div> </div> </div>
Use the attribute [data-iframe="/path/to/file.html"] to import a html.
<div class="demo"> <div class="demo-item" data-iframe="/path/to/file.html"></div> </div>
You can use wrap multiple .demo-item.
<div class="demo"> <div class="demo-item"> <div class="demo-source" data-lang="html"> <!-- code here --> </div> </div> <div class="demo-item demo-preview"> <div class="demo-source" data-lang="html"> <!-- code here --> </div> </div> <div class="demo-item" data-iframe="/path/to/file.html"></div> </div>
Use the attribute [data-name="tab name"] to give a name to the tabs.
<div class="demo"> <div class="demo-item" data-name="tab name"> <div class="demo-source" data-lang="html"> <!-- code here --> </div> </div> <div class="demo-item" data-name="another tab name"> <div class="demo-source" data-lang="html"> <!-- code here --> </div> </div> </div>

Usage

Generic usage for all javascript components.

All components are initialized automatically on document ready, if you want to disable it and initialize manually just set this variable before document ready $.fn.xtInitManual = true;.

Just assign the attribute data-xt to an element. It is initialized with defaults settings, you can specify settings in the attribute with a json format: data-xt='{"class": "custom-class", "target": "#custom-target", "min": 1, "max": 3}'.

You can initialize any element with component data- attributes via javascript:


// init only $(element)
$(element).xtInitAll();
// init $(element) and all descendants
$(element).xtInitAll(true);

You have the option to initialize manually in javascript this way:


$('[data-xt]').xt();
$('[data-xt-ajax]').xtAjax();
$('[data-xt-toggle]').xtToggle();
$('[data-xt-collapse]').xtCollapse();
$('[data-xt-menu]').xtMenu();
$('[data-xt-scroll]').xtScroll();

Manual initialization arguments are the same as json data- arguments:


$(this).find('[data-xt]').xt({
  "class": "custom-class",
  "target": "#custom-target",
  "min": 1,
  "max": 3
});

Toggle

Generic javascript component to toggle a class. You can customize the class, the event, the target, the min and max activations.

Defaults

By default it toggles the class .active on click.
By default it has min: 0 and max: 1.

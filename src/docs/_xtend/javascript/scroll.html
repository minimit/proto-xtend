Usage

Markup

Markup initialization of scroll via markup.

The scroll plugin adds to the "element" and body the .scroll class when the windows is scrollTop reach the element.


<div data-xt-scroll></div>

The plugin clones the "element" and hides it, so it adds the space of the "element" also if it's position: fixed;, it works with responsive automatically.

You can set custom class, start position with scrollTop, end position with scrollBottom.


<div data-xt-scroll='{"class": "scroll-custom", "scrollTop": ".element", "scrollBottom": ".element"}'></div>

You can use the scss mixin to style the scroll "element".


// html default class .scroll, default z-index 70
@include xt-scroll('.card[data-xt-scroll]');
// html custom class .scroll-card, default z-index 70, absolute when not scrolling
@include xt-scroll('.card[data-xt-scroll]', '.scroll-card', 'absolute');

Demos

Just add [data-xt-scroll] to the "element". Elements inside the scrolling item with class .show-scroll will be shown only when scrolling, with class .hide-scroll will be hidden only when scrolling.
To have the "element" fixed or absolute when not scrolling, specify it in the mixin $start. The space is added automatically by the cloned "element".
By default the position on scroll is top: 0 if you want a bottom: 0 or a value, specify it in the mixin $position and $pos.
You can set custom class, start position with scrollTop, end position with scrollBottom.
Use the classes .scroll-show-up, .scroll-show-down, .scroll-hide-up, .scroll-hide-down to add animations when the element show or hide in the specific direction.

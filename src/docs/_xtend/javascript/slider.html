Usage

Initialization

Html Initialization


<div class="element" data-xt-slider'>
</div>
<div class="element" data-xt-slider='{"elements": ".item"}'>
  <div class="item"'>
  </div>
</div>


$('.element').xtInitAll(); // init only '.element'
$('html').xtInitAll(true); // init 'html' and all descendants

Js Initialization


$('.element').xtSlider();
$('.element').xtSlider({"elements": ".item"});
Important: the plugin's attributes must be a valid json format, so with double quotes inside.

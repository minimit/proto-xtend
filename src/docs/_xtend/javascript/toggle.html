Usage

Usage for all javascript components.

Automatic

Assign data-xt attribute to the element, this will extend the plugin's default settings with your own settings.


<button class="button color-primary" data-xt='{"target": ".collapse"}'>toggle</button>
Important: the plugin's attributes must be a valid json format, so with double quotes inside.

Manual (optional)

If you want to disable auto initialization just set $.fn.xt.manualInit = true; before document ready.

You can initialize all the elements:


$(element).xtInitAll(); // init only $(element)
$(element).xtInitAll(true); // init $(element) and all descendants

Or one by one:


$('[data-xt]').xt();
$('[data-xt]').xt({"target": "#collapse", "min": 1, "max": 3}); // arguments are the same as json data- arguments

Toggle

Generic javascript component to toggle a class. It's a subclass of data-xt so it behaves in the same way, it only have different default options.

Defaults

By default it toggles the class .active on click.
By default it has min: 0 and max: 1.

Height

You have the option to animate height on targets. You can change .xt-height animations in the theme.

Just add the class .xt-height to the "target". With theme's css .xt-height animation.
Target
.xt-height { -webkit-transition: height 0.3s cubic-bezier(.63,.36,0,.97); transition: height 0.3s cubic-bezier(.63,.36,0,.97); will-change: height; }

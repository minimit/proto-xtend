Usage

Initialization

Html Initialization


<div class="element" data-xt-toggle'>toggle</div>
<div class="element" data-xt-toggle='{"elements": ".button", "targets": ".collapse"}'>
  <div class="button"></div>
  <div class="collapse"></div>
</div>


$('.element').xtInitAll(); // init only '.element'
$('html').xtInitAll(true); // init 'html' and all descendants

Js Initialization


$('.element').xtToggle();
$('.element').xtToggle({"elements": ".button", "targets": ".collapse"});
Important: the plugin's attributes must be a valid json format, so with double quotes inside.

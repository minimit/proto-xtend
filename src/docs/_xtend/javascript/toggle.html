Usage

Setup

Html Initialization


<div class="element" data-xt-toggle'></div>
<div data-xt-toggle='{"elements": ".element"}'>
  <div class="element"></div>
  <div class="toggle-block"></div>
</div>
Important: the plugin's attributes must be a valid json format, so with double quotes inside. 

$('.element').xtInitAll(); // init only '.element'
$('body').xtInitAll(true); // init 'html' and all descendants

Js Initialization


$('.element').xtToggle();
$('div').xtToggle({"elements": ".element"});

Examples

Single

You can toggle a single Element with data-xt-toggle. The toggled class is always .active. You can specify targets by #id.

For the targets use the classes .toggle-block, .toggle-inline-block, .toggle-inline, .toggle-none. These classes toggle the display property on .active (@TODO link utils classes).

Usage

Html Initialization


<div class="element" data-xt-toggle'></div>
<div data-xt-toggle='{"elements": ".element"}'>
  <div class="element"></div>
  <div class="toggle-block"></div>
</div>
Important: the plugin's attributes must be a valid json format, so with double quotes inside. 

$('.element').xtInitAll(); // init only '.element'
$('body').xtInitAll(true); // init 'html' and all descendants

Js Initialization


$('.element').xtToggle();
$('div').xtToggle({"elements": ".element"});

For the targets use the classes .toggle-block, .toggle-flex, .toggle-inline, .toggle-inline-block, .toggle-none. These classes toggle the display property on .active.

Manager

You can create a "manager" by assigning elements, by default the targets are all children with a class that starts with .toggle-, elements and targets must be inside the manager.

With multiple targets, the ones activated is the one with the same index of the toggled element. Nested targets are filtered out.

Min and max

Inside a manager you can specify min or max activations. Default is "min": 0 and "max": 1. Use "max": "Infinity" for no limits.

The required min activation is fulfilled on initialization. Automatic [disabled] on reaching the min value.

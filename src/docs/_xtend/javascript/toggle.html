Usage

Settings

Html settings


<button class="button" data-xt-toggle'>
  toggle
</button>
<div data-xt-toggle='{"elements": ".button", "targets": ".collapse"}'>
  <button class="button">
    toggle
  </button>
</div>


$('.button, div').xtInitAll(); // init only '.button'
$('html').xtInitAll(true); // init 'html' and all descendants

Js settings


$('.button').xtToggle();
$('.div').xtToggle({"elements": ".button", "targets": ".collapse"});
Important: the plugin's attributes must be a valid json format, so with double quotes inside.

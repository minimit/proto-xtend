Usage

Automatic

Automatic initialization of ajax component.

Just assign $.fn.xt.automaticAjax before document ready, 'baseurl' is the prefix of links that have ajax activated ('baseurl': '/' activates ajax on all links that starts with '/'), 'target' is the document selector to replace on each ajax call.


$.fn.xt.automaticAjax = {
  'baseurl': '/',
  'target': '.site-wrapper',
};

If you want to reinit ajax use this function $.fn.xt.initAjax();

Manual

Manual initialization of ajax component.

You can call xtAjax manually on the links you want to ajaxify.


var ajaxBaseUrl = '/';
var ajaxTarget = '.site-wrapper';
$('a[href^="' + ajaxBaseUrl + '"]').xtAjax({"target": ajaxTarget});
$(ajaxTarget).on('ajax.done.xt', function(e, obj, $data) {
  $(this).find('a[href^="' + ajaxBaseUrl + '"]').xtAjax({"target": ajaxTarget});
  $(ajaxTarget).xtInitAll(true); // reinit xtend
});

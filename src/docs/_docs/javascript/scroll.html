Usage

Html Initialization


<div class="element" data-xt-scroll></div>
<div class="element" data-xt-scroll='{"top": "#top", "bottom": "#bottom"}'></div>
Important: the plugin's attributes must be a valid json format, so with double quotes inside. 

$('.element').xtInitAll(); // init only '.element'
$('body').xtInitAll(true); // init 'html' and all descendants

Js Initialization


$('.element').xtScroll();
$('.element').xtScroll({"top": "#top", "bottom": "#bottom"});

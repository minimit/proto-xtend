Usage

Html Initialization


<body data-xt-ajax='{"elements": "a[href^=\"/\"]", "targets": ".site-wrapper"}'></body>
Important: the plugin's attributes must be a valid json format, so with double quotes inside. 

$('.element').xtInitAll(); // init only '.element'
$('body').xtInitAll(true); // init 'html' and all descendants

Js Initialization


$('body').xtAjax({'elements': 'a[href^="/"]', 'targets': '.site-wrapper'});

ajax.populated.xt

If you need to reinit your js on ajax, listen to the ajax.populated.xt event.


$('.site-wrapper').on('ajax.populated.xt', function(e, obj, $data) {
  main($(this)); // custom function on ajax
});
